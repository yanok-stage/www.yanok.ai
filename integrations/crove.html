Crove Integrations

Crove allows you to create repetitive business documents as intelligent templates that use variables and logical conditions.

Triggers

Document Completed
Triggers when a document is completed.
Document Created
Triggers when a new document is created.
Document Submitted
Triggers when a document is submitted.

Actions

Complete Document
Mark the document as completed
Create Document
Create a new document.
Create Invitation Link
Create invitation link to fill or sign the document.
Send Document
Send email invitation link to fill & sign the document.
Generate PDF From Document
Generate PDF of a document and return PDF URL.
Generate Document PDF From Template
Generate PDF of a document created from the template
Get Document Details
Get details of a document. Example: Name, Current Status, etc.
Submit Document
Submit the document like you do it via Crove form.
Update Document Details
Update details of a document. Example: Name, Current Status, etc.
Update Document
Update values of variables of a document.
