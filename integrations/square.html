Square Integrations

Square creates tools that help sellers of all sizes start, run and grow their businesses. Square’s point-of-sale service offers tools for every part of running a business, from accepting credit cards and tracking inventory to real-time analytics and invoicing.

Triggers

New Appointment
Triggers when a new appointment is created.
New Catalog Item
Triggers when a new catalog item is created.
New Customer
Triggers when a new customer is created.
New Invoice
Triggers when a new invoice is added.
New Order
Triggers when a new order is processed.
New Payment
Triggers when a new payment is created.

Actions

Add Customer to Group
Adds a customer to a group.
Create Appointment
Creates a new appointment.
Change Inventory
Adjusts an inventory or changes its physical count.
Create Customer
Creates a customer.
Create Invoice
Creates a new draft invoice for an order.
Create Paid Order
Creates an externally paid and completed order.
Refund Payment
Refunds a completed payment.
Remove Customer From Group
Removes a customer from a group.
Send Invoice
Sends an existing draft invoice.
Update Customer
Updates a customer.
Find Customer
Finds a customer based on email, phone number, or reference ID.
Find Service
Finds a service.
Find Order
Finds an order based on customer, fulfillment status, or order state.
Find Customer
Find a customer. Optionally, create one if none are found.
Find Order
Find an order. Optionally, create one if none are found.
