Repsly Integrations

Repsly's retail execution platform provides data-driven merchandising, field sales, and promotion execution solutions to help CPG brands and merchandisers maximize the impact of their field teams.

Triggers

New Audit With Items
Triggers when a new audit is created. Returns audit items with full headers.
New Client Status Change
Triggers when a client status is updated.
New Client With Details
Triggers when a new client is created.
New Form With Details
Triggers when a new form is created.
New Order
Triggers when a new order is created.
New Item in Order
Triggers when a new order item is added to order. Returns order items with full headers.
New Schedule
Triggers when a new schedule is created.
New Missed Schedule
Triggers when a new schedule is missed.
New Visit Start
Triggers when a new visit is created.
New Visit End
Triggers when a visit ends. Available only when the visit was explicitly ended on the mobile device.
New Workday End
Triggers when representative ends a workday.
New Workday Start
Triggers when representative starts a new workday.
