Microsoft To Do Integrations

Microsoft To Do is an intelligent task management app that makes it easy to plan and manage your day. With its intelligent Suggestions, To Do removes all the clutter and empowers you to focus on what's important, when it's important.

Triggers

Completed Task
Triggers when a task is completed.
New List
Triggers when a new list is created.
New Task
Triggers when a new task is created.
Updated Task
Triggers when a task is updated.

Actions

Create List
Creates a new list.
Create Task
Creates a new task.
Complete Task
Marks a task as complete.
Find a Task
Search for a Task.
Find or Create Task
Finds a task by title, or creates one if none is found.
