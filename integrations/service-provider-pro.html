Service Provider Pro Integrations

SPP is a shopping cart and project management software for selling productized services.

Triggers

Invoice Paid
Triggers when an invoice has been paid.
New Client
Triggers when a new client is created.
New Invoice
Triggers when a new invoice is created (manually or during checkout).
New Order
Triggers when a new order is created.
New Ticket
Triggers when a new ticket is created.
Order Data Submitted
Triggers when a client submits project data.
Order Status Changed
Triggers when the status of an order is changed.

Actions

Assign Order
Assigns order to a team member.
Send Message
Sends a message to the client or team.
Add Tag
Adds a tag to an order.
Unassign Order
Unassign order from a team member.
Update Status
Updates order status.
