Shopping Feed Integrations

Shopping Feed is a single platform to List, Sync, and Manage your products, inventory and orders with the world's largest marketplaces, adspaces, affiliate channels and shopping engines.

Triggers

Failed Catalog Import
Triggers when a catalog has failed being updated.
Failed Export
Triggers when an export has failed being updated.
Failed Order Acknowledgement
Triggers when an order has failed being acknowledged by your store.
Failed Order Shipment
Triggers when an order has failed being shipped.
Order
Triggers when an order is pulled from marketplace.

Actions

Accept Order
Notify Shopping Feed that an order has been accepted.
Acknowledge Order Import
Notify Shopping Feed that an order has been imported in your store.
Ship Order
Notify Shopping Feed that an order has been shipped.
Unacknowledge Order Import
Notify Shopping Feed that an order is no more imported in your store.
Update Product Inventory
Update product inventory.
