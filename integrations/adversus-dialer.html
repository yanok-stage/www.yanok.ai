Adversus Dialer Integrations

A web-based auto dialer solution used by professionals to streamline outbound outreach by phone.

Triggers

New Appointment
Triggers when a new appointment has been booked
Lead Updated
Triggers when a lead is updated/saved in Adversus.

Actions

Add Contact to Campaign
Adds a specific contact to a specified campaign.
Create Contact
Creates a new contact in a specified pool.
Create Lead
Creates a new lead with a specified pool and campaign.
Update Contact
Updates an existing contact.
Update Lead
Updates lead attributes (campaign, status, next contact time) based on an lead ID
Find Contact
Lookup existing contacts in Adversus, e.g. to avoid duplicates. The first result is used if multiple are found.
Find Lead
Finds a lead based on the ID
Find User by Email
Finds a user based on their email
Find User by ID
Finds a user based on their Adversus user ID
