Vonage Voice API Integrations

The Vonage Voice API (formerly Nexmo) allows you to make and receive calls with text-to-speech, audio playback, and forwarding.

Triggers

New Conference Call
Triggers when a new call is received and joined into a conference.
New Forwarded Call
Triggers when a new call is received and forwarded to a number.
New Stream Call
Triggers when a new call is received and an audio file is played.
New TTS Call
Triggers when a new call is received and a message is read.

Actions

Create Conference Call
Creates a call and joins the user into to a conferance
Create Stream Call
Creates a call and plays the user an audio file
Create Text to Speech Call
Creates a call and reads a message
Create Two-Way Call
Creates a call between 2 numbers
