Helpy Pro Integrations

Helpy Pro is the customer support platform engineered for companies that are serious about customer data security and control.

Triggers

Agent Replies to Ticket
Triggers when an agent replies to a ticket.
Customer Replies to a Ticket
Triggers when a customer replies to a ticket.
New Ticket Created
Triggers when a new ticket is created.
New User Created
Triggers when a new user is created.
Ticket Assigned to an Agent
Triggers when a ticket has been assigned to any agent.
Ticket Tagged With
Triggers when a ticket is given a specific tag.

Actions

Create Reply to Ticket
Reply to a given ticket. Can be used to create a reply or note, written by either the customer or an agent.
Create Ticket
Creates a New Ticket.
Create User
Create a user in Helpy.
Tag Ticket
Tag a ticket with a provided tag.
Update User
Update a customer or agent user in Helpy
Find Ticket
Find a ticket using either the ticket number, the subject, or the email address of the user who opened the ticket. If the user has multiple tickets in the system, the most recent will be used.
Find User
Find a user in Helpy by email address, ID, or full name.
Find or Create User
Find a user in Helpy by email address, ID, or full name
