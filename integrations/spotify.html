Spotify Integrations

Spotify is a music streaming service that provides digital content from record labels and media companies.

Triggers

New Playlist
Triggers when you create a new playlist.
New Track Added to Playlist
Triggers when a new track is added to one of your playlists or a playlist you follow.
New Saved Track
Triggers when you save a new track to "Your Music" library.

Actions

Add Track to Playlist
Adds a track to one of your playlists.
Create Playlist
Creates a new playlist.
Save Track
Save a track to "Your Music" library.
Find Track
Look up a track by title/artist or by entering a track ID.
Get Audio Features for a Track
Gets the audio features (duration, tempo, etc.) for a track.
