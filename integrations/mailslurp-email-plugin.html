MailSlurp Email Plugin Integrations

MailSlurp is an API to generate test email accounts for sending and receiving real emails in tests.

Triggers

New Attachment
Triggers when a new email attachment is received.
New Contact
Triggers when a new contact is send in received emails.
New Email
Triggers when a new email is received.
New Inbox
Triggers when a new inbox is created.

Actions

Create Contact
Create a new email address
Send Email
Send a new email from a MailSlurp inbox. Create an inbox first if one does not exist.
Create Inbox
Create a new email address
