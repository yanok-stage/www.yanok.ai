Nexkey Integrations

Nexkey, enabling industry leading and cost effective access control.

Triggers

Key Share on Lock
Triggers when a key share takes place on specified lock.

Actions

Add Group Members
Adds members to specified group
Perform Unlock
Perform an unlock on target lock. Only for an Internet-connected Controller.
Remove Group Members
Removes members from specified group
Revoke Keys
Revoke a key from a target user.
Send Keys
Share an Admin or User level key for a lock to a user. This key can be scheduled to only allow access within certain time restrictions or it can be unrestricted and live indefinitely.
Find Group
Find a group by name.
Find Lock
Find a lock by its name or description.
Find User
Find a user by first name, last name, email or phone.
