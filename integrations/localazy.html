Localazy Integrations

Localazy is a highly automated localization platform allowing anyone to translate their products and content into multiple languages easily.

Triggers

Comment Added
Triggers when a comment is added to a project.
Content Imported & Content Imported Empty
Triggers when importing to a project is finished (from CLI or Gradle or File Upload).
Project Published
Triggers when a project is successfully published (applies to tags too).

Actions

Import to Localazy
Import source and other languages keys to Localazy.
Zip Files in Languages
Allows you to select from the project's files and languages and bundle them into a zip file.
