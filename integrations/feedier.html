Feedier Integrations

Feedier lets you collect valuable feedback from your users by providing an enjoyable & rewarding experience.

Triggers

New Feedback
Triggers when a new feedback is received.
New Item Tagged
Triggers when an item is tagged.
Segment
Triggers when a new segment is created. Do not use it.

Actions

Import Feedback
Import a new feedback with a given set of answers.
Send Email
Sends a new email to ask gently the user to give his feedback.
Send SMS
Sends a SMS text message to ask gently the user to give his feedback.
Update Feedback Status
Update a feedback status and/or its owner.
