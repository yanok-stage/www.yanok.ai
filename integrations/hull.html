Hull Integrations

Hull is a customer data platform that unifies your customer data in a single place and sends that data to all of your connected tools.

Triggers

Account Attribute Updated
Triggers when an account attribute is updated.
Account Created
Triggers when an account is created.
Account Entered Segment
Triggers when an account enters a segment.
Account Left Segment
Triggers when an account leaves a segment.
User Attribute Updated
Triggers when a user attribute is updated.
User Created
Triggers when a user is created.
User Entered Segment
Triggers when a user enters a segment.
User Event Created
Triggers when a user event is created.
User Left Segment
Triggers when a user leaves a segment.

Actions

Create or Update an Account
Sends Attribute updates to the account identified by a domain. Will create the account if not created already.
Create or Update a User
Sends Attribute updates to the user identified by an email. Will create the user if not created already.
Create User Event
Adds an Event to the user identified by an email. Will create the User if not created already
Find an Account
Search for an Account by domain or external_id
Find a User
Search for a User by email or external_id
