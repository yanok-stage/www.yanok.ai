MyCarTracks Integrations

MyCarTracks is a reliable smartphone-based vehicle tracking service

Triggers

Tracks
Triggers when a new track is available.
Users List
Triggers when a new user/driver is created. Contains the list of users that are connected to you, either a driver or any of user you created.
Vehicles List
Triggers when a new vehicle is created. Contains a list of all vehicles. The response includes also vehicles that are not included in your paid plan. Use parameter 'paid' to filter paid or not-paid vehicles.

Actions

Find One Track With Data
Returns track details identified by an ID, optionally including track points in the requested format (KML, CSV, GPX).
Find Tracks
Finds Tracks by the filter
