DynaPictures Integrations

Design image templates and auto-generate personalized images via API.

Triggers

New Image
Triggers when a new image is generated from a template.
New Template
Triggers when a new image template is created in DynaPictures.
New Template With Layer Params
Triggers when a new image template is created. Provides a list with the names of layer params that need to be specified in oder to generate an image based on this template. This list of param names is useful to create columns in a spreadsheet and use it to generate actual images later.

Actions

Create Image
Generates a new image based on an image template.
