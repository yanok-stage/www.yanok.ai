Isofy Integrations

Isofy is a network management platform that allows you to control your co-working space's network infrastructure.

Triggers

New Member
Triggers when a new Member is created in Isofy.

Actions

Inactivate Member
Marks a member as inactive, removing network access from their devices and preventing network login.
Update or Create a Member
Update an existing member, or create a new member if one does not already exist
