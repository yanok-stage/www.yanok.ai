Splash Integrations

Splash is an event marketing automation platform that empowers brands to efficiently scale event programs, reach and connect with target audiences in-person, and connect event-driven engagement to business results.

Triggers

Attendee Added
Triggers when someone is manually added to a guest list.
Attendee Check-In
Triggers when an attendee is checked in to an event.
Attendee Waitlisted
Triggers when someone is added to the wait list.
New Attendee
Triggers when someone RSVPs to an event.
Ticket Purchased
Triggers when a ticket is purchased.

Actions

Add Attendee
Adds a new attendee to an event.
Update Attendee
Updates an existing attendee. If attendee doesn't exist, the attendee is added.
