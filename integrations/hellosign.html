Dropbox Sign Integrations

Dropbox Sign is a brand new way to collect signatures and send important documents from an easy web interface. Just upload a PDF or Word Document and make your templates, and send them to anyone!

Triggers

Signature Request Declined
Triggers when a signature request is declined (requires a paid API plan for non test_mode requests).
Signature Request Sent
Triggers when a signature request is sent (requires a paid API plan for non test_mode requests).
Signature Request Signed
Triggers when a signature request is signed (requires a paid API plan for non test_mode requests).
Signature Request Viewed
Triggers when a signature request is viewed (requires a paid API plan for non test_mode requests).

Actions

Send Signature Request From Template
Sends a HelloSign template out for signature request.
