Colligso TextIn Integrations

Colligso enables businesses to grow profitably by providing an easy to use and integrated platform to create and cultivate digital customer relationships. TextIn helps businesses by engaging with customers directly using text (SMS)/ picture (MMS) messaging. App-less, no app installation required by your customers. Easy opt-in and opt-out. Universal, available on every mobile phone where SMS is available.

Triggers

Actions

Get Permission
Opt-in for text marketing.
Create Customer
Import Customer from other app to your Colligso account. Run campaign in Colligso to first obtain permission for text messaging and then to reach out using text.
Send Message
Sends a message (SMS)
