Process Street Integrations

Process Street is a modern process management platform for teams. We help teams share their core processes, then transform them into powerful no-code workflows.

Triggers

Data Set Record Created
Triggers when a new record is added to a data set.
Data Set Record Deleted
Triggers when an existing record is deleted from a data set.
Data Set Record Updated
Triggers when an existing record is updated in a data set.
New Attachment
Triggers when someone attaches a file to a workflow run.
New Workflow Run
Triggers when a workflow is run.
New Comment
Triggers when someone posts a comment on a workflow run.
New Task Checked
Triggers when a task is checked off in a workflow.

Actions

Run Workflow
Runs a workflow.
Create Data Set Record
Adds a new record to a data set.
Delete Data Set Record
Deletes an existing record in a data set.
Update Workflow Run
Update an existing workflow run.
Update Data Set Record
Updates an existing record in a data set.
Find Data Set Records
Search for data set records by form fields.
Find Workflow Run
Search for a workflow run by ID, name, form fields, and other attributes.
Create Data Set Record if it doesn't exist yet
Creates a data set record if none were found by the search.
Run Workflow if it doesn't exist yet
Creates a workflow run if none were found by the search.
