ChatBeacon Integrations

ChatBeacon is a live chat solution for your website, allowing you to monitor your website traffic in real-time, provide customer service and support or capture sale leads. ChatBeacon offers three chat channels: website, SMS and Facebook Chat.

Triggers

Chat Ended
Triggers when a chat ends.
New Chat Request
Triggers when a new chat request has been made.
New Session
Triggers when a new session is created.
Survey Submitted
Triggers when an offline or post chat survey is submitted.
Session on Page or Query
Triggers when visitor lands on page/query.
Page Visits
Triggers when a session has visitor a number greater than or equal to the specified value.
Chat Segment Contains
Triggers when a chat segment contains the specified data.
Site Variable Equals
Triggers when a session with a site variable of equal value.
