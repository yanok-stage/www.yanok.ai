Zoho People Integrations

Zoho People is an online HR management software which helps you automate all your HR processes efficiently.

Triggers

New Event
Triggers based on the date field of a form.
New Record
Triggers when a record is added in a specific form.

Actions

Add Company File
Adds a new file as a Company File in the Files module.
Add HR File
Adds a new file as HR file in the Files Module.
Add Record
Adds a new record in the selected form.
Create Attendance Entry
Creates a Check-in and/or Check-out entry.
Upload a File in a Form
Uploads a file in the 'File Upload' or 'Image' field of the selected form.
Get Employee
Lookup an employee by Email ID or Employee ID.
