CXera Integrations

CXera enables “one-click” creation of microsites that help you collaborate, track, take action and continuously improve customer success.

Triggers

New Experience Found
Triggers when a new Experience is created.
Experience Visited
Triggers when a user visits an Experience.
Experience Item Downloaded
Triggers when a user downloads an item from Experience.

Actions

Clone a Default Experience
Creates a new Experience by cloning a default Experience
Add User to Experience
Adds a user to an Experience
