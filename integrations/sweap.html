Sweap Integrations

Automate time-consuming tasks that remove focus from the impact of your event. For in-person & online events.

Triggers

New Event Created
Triggers when a new Event is created in your account.
New Guest Added to Event
Triggers when a new Guest is put on the Guest list of a specific event in Sweap.

Actions

Create Guest for Event
Create a Guest for an Event.
Update Guest
Finds a guest by ID and updates limited fields
Find Event by ID
Finds an Event in Sweap by ID.
Find Guest by ID
Finds a Guest resource by ID.
Search Event
Finds an Event in Sweap that best matches your filters.
Search Guest
Finds a Guest that best matches your filters for an Event. If nothing found, you can create a new Guest.
Find (by ID) or Create Guest
Get a guest resource by id.
Find or Create Guest
Finds a Guest that best matches your filters for an Event. If nothing found, you can create a new Guest.
