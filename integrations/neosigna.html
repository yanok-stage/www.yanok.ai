neosigna Integrations

neosigna is a collaborative contract management platform to speed up creating contracts and getting them signed.

Triggers

New Contract Created
Triggers when a new contract is created.

Actions

Create Contract
Creates a new Contract (and its Draft version).
Create Counterparty
Creates a new counterparty.
Create Draft
Creates a new Draft version for a Live Contract.
Create User
Creates a new user and automatically invite him/her to setup his/her account.
Update Component
Update the value of a component based on its internal id.
Get Component
Get a component (including its internal id) from a given Contract version, based on a Component code.
Get Contract
Returns a Contract matching a Contract Code.
Get Latest Version of Contract
Get the latest version of a Contract based on a Contract internal id. Draft if it exists, Live otherwise.
