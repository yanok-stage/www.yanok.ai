PCRecruiter Integrations

PCRecruiter is a CRM that serves customers worldwide doing permanent placement, 3rd party recruitment, corporate direct hiring, and staffing/contracting.

Triggers

New Name in Rollup List
Triggers when a candidate is added to the rollup list specified.
New Placement
Triggers when a new contract temp or permanent placement is made inside of PCRecruiter.
New Position in Rollup List
Triggers when a position is added to the rollup list specified.
Pipeline Stage Reached - PCR10 Only
Triggers when a candidate reaches a particular stage of a pipeline. This trigger is only compatible with PCR10.

Actions

Create Company Record
Creates a Company Record in PCRecruiter.
Create Name Activity
Creates a name activity inside PCRecruiter
Create Name Record
Creates a Name Record in PCRecruiter.
Create Position Activity
Creates a position activity inside PCRecruiter.
Create Position Record
Creates a Name Record in PCRecruiter.
Update Name Record
Updates an existing name record in PCRecruiter.
