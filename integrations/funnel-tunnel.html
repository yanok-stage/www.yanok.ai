Funnel-Tunnel Integrations

Funnel-Tunnel is all-in-one software for small and medium-sized companies to successfully digitize. From design to marketing to sales, Funnel-Tunnel is a complete solution.

Triggers

Contact Updated
Triggers when an existing contact updated.
New Form Submission
Triggers when a form submitted.
New Order Created
Triggers when a new order created.

Actions

Create Contact
Create or update a contact. Unique identifier is email.
Create Product
Creates a new product in your store.
Update Product
Updates an existing product. Replaces only data that is set.
Find Contact
Find contact by email. Optionally, create one if none is found.
Find Product
Find product by ID. Optionally, create one if none is found.
