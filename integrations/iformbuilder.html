iFormBuilder Integrations

iFormBuilder is a universal, cloud based mobile data collection ("forms") platform with an emphasis on security on mobile devices, and collecting data with or without an Internet connection.

Triggers

New Company Profile
Triggers when a new company is created.
New Element
Triggers when a new element is created on a form.
New Form
Triggers when a new form is created in a profile.
New Record
Triggers when a new record is created in a form.
New User
Triggers when a new user is created in a profile.

Actions

Create Element
Creates a new element.
Create Form
Create a new form in a profile.
Create Record
Creates a new record.
Update Form Assignment
Update form permissions.
Update Record
Updates an existing record.
Update Record Assignment
Assign a record to a specific user.
Create User
Create a new user in a profile.
