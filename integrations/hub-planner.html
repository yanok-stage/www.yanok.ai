Hub Planner Integrations

Hub Planner is a resource planning and scheduling service with timesheets and dynamic reporting.

Triggers

New Booking
Triggers when a new booking is created.
New or Updated Project
Triggers when a project is created or updated.
New or Updated Resource
Triggers when a resource is created or updated.
New or Updated Time Entry
Triggers when a new time entry is created or updated.

Actions

Create Booking
Create / Schedule a booking for resource and project or event.
Create Event
Creates a new event.
Create Milestone
Create a Milestone for a Project.
Create Project
Creates a new project.
Create Resource
Creates a new resource.
Create Time Entry
Create timeEntry for resource and project.
Update Booking
Updates an existing booking.
Find Booking
Find Booking by id.
Find Event
Find event by name or id.
Find Project
Find project by name, code or id.
Find Resource
Find resource by it's ID, name or email address.
Find TimeEntry
Find timeentry by id.
