Vortext Integrations

vorText is an ecosystem of SMS Apps you can use to engage with your customers in new exciting ways.

Triggers

New Contact
Triggers when a new contact is ceated
New vorText Appointment/Event
Triggers when a new appointment or event is created. For example, when a person books and appointment or registers for an event.

Actions

Create New Appointment
Creates a new appointment in vorText
Create New Contact
Creates a new vorText contact and optionally adds them to a list
Send a JSON String
Sends a JSON package to the end-point for processing
Send a SMS Message
Sends an SMS message
