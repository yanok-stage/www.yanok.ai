CRM Web App Integrations

CRM Web App is a Zapier App that interacts with Credit Money Machine Web, CRM Business Companion, CRM Monitor Companion, and Automatic Business Machine.

Triggers

New Affiliate
Triggers when a new affiliate is added or when an existing client is assigned as an affiliate.
New Client
Triggers when a new client is added or when an existing client is assigned as a client.
New Not Active Client
Triggers when a not active client is added or when an existing client is assigned as a not active client.
New Lead
Triggers when a new lead is added or when an existing client is assigned as a lead.

Actions

Create Affiliate
Create an affiliate into the program
Create Client
Adds a new client into the program
Create Lead
Creates a new lead into the program
