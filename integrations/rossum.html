Rossum Integrations

Rossum is cloud-based app that uses artificial intelligence to replicate the human approach to data extraction from business documents.

Triggers

Document Export
Triggers when a document is exported in Rossum.

Actions

Export Confirmed Documents
Moves all confirmed documents to exported state. This action does not output any data, use "Export a Document" trigger for processing of exported data. Note: This action requires "Confirmed State" to be enabled on target queue.
Upload a Document
Uploads a document to one of Rossum queues
