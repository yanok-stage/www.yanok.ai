Guru Integrations

Guru is a collaborative knowledge management solution where answers find you—everywhere, every time. Guru connects teams and people to one another and surfaces the know-how of those who know best, and turns collections of information into useful knowledge. By delivering the right knowledge at the right time, Guru reduces new-hire ramp time, drives amazing customer experiences, and makes teams more collaborative and productive.

Triggers

Announcement Sent
Triggers when a new announcement is sent
New Board
Triggers when a new Board is created.
Card Archived
Triggers when a user archives a Card.
Card Comment Created
Triggers when a user comments on a Card.
Card Content Copied
Triggers when a user either copies the content within the Card or clicks on the “Copy Card content" icon.
Card Created
Triggers when a new Card is published.
Card Favorited
Triggers when a user favorites a Card.
Card Link Copied
Triggers when a user clicks on the "Copy Card link" icon.
Card Unfavorited
Triggers when a user removes a Card from their "My Favorite Card" list.
Card Unverified
Triggers when a Card becomes unverified.
Card Updated
Triggers when a user makes an edit to a Card.
Card Verified
Triggers when a user verifies a Card.
Card Viewed
Triggers when a user views a Card.
New Collection
Triggers when a new Collection is created.
Collection Trust Score Crosses Threshold
Triggers when a Collection’s Trust Score crosses a specified threshold.
New Group
Triggers when a new Group is created.
New Tag
Triggers when a new Tag is created.
Team Card Count Reaches a Milestone
Triggers when a Team’s Card count reaches a milestone.
User Added to Group
Triggers when a user is added to a Group.

Actions

Add Tag to Card
Adds an existing Tag to a Card.
Add User to Group
Adds a user to a Group.
Archive Card
Archives a Card.
Create Card
Creates a new Card.
Create Group
Creates a new Group.
Invite User
Adds a user to your Guru Team.
Create PDF From a Guru Card ID
Generate a PDF file from a Guru card ID. You must have access to the Guru card in order to generate a PDF
Remove User
Removes a user from your Guru Team.
Remove User From Group
Removes a user from a Group.
Restore Card
Restores a previously Archived Card.
Unverify Card
Unverifies a Card.
Verify Card
Verifies a Card.
Find Card
Finds a Card. If you cannot find the Card, you can create a Card
Find Archived Card
Finds an Archived Card.
Find Tag
Finds a Tag.
Find or Create Card
Finds a Card. If you cannot find the Card, you can create a Card
