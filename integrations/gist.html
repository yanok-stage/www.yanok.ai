Gist Integrations

Gist is an all-in-one email marketing automation, live chat & help desk platform.

Triggers

Subscribed to a Campaign
Triggers when a contact is subscribed to a campaign.
New Contact
Triggers when you add a new contact.
Tag Added to Contact
Triggers when a specific tag gets added to a contact.
Performed an Event
Triggers when a contact performs an event.
Unsubscribed From a Campaign
Triggers when a contact is unsubscribed from a campaign.
Tag Removed From Contact
Triggers when a specific tag gets removed from a contact.
Updated a Contact Property
Triggered every time a specified property is updated on any contact. This will also be triggered when a contact is created with this property.

Actions

Add Tag to an Existing Contact
Apply a tag to an existing contact.
Create or Update Contact
Creates a new Contact or updates an existing one.
Remove Tag From Existing Contact
Removes a tag from an existing contact.
Subscribe to a Campaign
Subscribes a contact to a specific campaign.
Unsubscribe From a Campaign
Unsubscribes a contact from a specific campaign.
Find a Contact
Finds an existing contact.
