Stormboard Integrations

An online stickynote whiteboard making meetings, brainstorms, and creative projects more productive and effective.

Triggers

Comment Created
Trigger when an new comment was posted on an idea.
Idea Created
Trigger when an idea was added to a storm.
Idea Moved
Trigger when an idea was moved to a different section of a storm.
Legend Change
Trigger when an idea's legend changes in a storm.
New Storm
Trigger when a new storm is created.
Task Updated
Trigger when any task change is made to an idea.

Actions

Create Idea
Creates a new Idea in an existing Storm.
Create Storm
Creates a new Storm.
Find Storm
Finds an existing storm by title.
Find or Create Storm
Finds an existing storm by title.
