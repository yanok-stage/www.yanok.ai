sms77 Integrations

Sms77 allows you to send a SMS to any mobile phone worldwide.

Triggers

Incoming SMS
Triggers when you have received a new incoming SMS.
Incoming Call
Triggers when somebody calls you.

Actions

Lookup CNAM
Query the Caller ID name for any phone number worldwide. Please note that not every phone number has an entry. Since the data is collected from various sources you could get geographical locations, company names/names or the network operator in response.
Lookup HLR
Looks up information regarding home location register for a given phone number.
Lookup MNP
Looks up information regarding mobile number portability for a given phone number.
Lookup Number Format
Query the national and international format of any mobile phone number. In addition, you will receive the corresponding network operator for the phone number as information (without porting information).
Register Webhook
Registers a Webhook.
Send SMS
Send a SMS to one or more specific number(s). Please take care that you should have enough balance on your sms77.io account.
Send Voice
Converts text to speech, calls the given number and reads it loud.
Get Analytics
Get detailed statistics of your account directly through our API.
Get Journal
You can use this endpoint to query the last entries of your logbooks. Each query returns a maximum of 100 entries.
