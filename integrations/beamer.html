Beamer Integrations

Beamer is an easy to use newsfeed and changelog for your website or app. Use it to announce news and features, and improve user engagement.

Triggers

New Feature Request Comment
Triggers when a new comment is sent for a feature request.
New Feature Request
Triggers when a new feature request is created.
New Feature Request Vote
Triggers when a new vote is sent for a feature request.
New Comment
Triggers when a new comment is sent for a post.
New NPS Response
Triggers when a new NPS response is received.
New Post
Triggers when a new post is published.
New Reaction
Triggers when a new reaction is sent for a post.

Actions

Create Post
Creates a new post.
