Cyberimpact Integrations

Cyberimpact is a CASL-Compliant email marketing service that makes it easy to create and send beautiful and efficient newsletter campaigns.

Triggers

New Group
Triggers when a new group is added.
New Group Member
Triggers when a new group member is added.
New Member
Triggers when a new member is added.

Actions

Add a New Group
Add a new static group in your account.
Add a Member to Groups
Adds a specific member to selected groups.
Add a New Member
Add a new member in your account. If you need the member to confirm its subscription, it is recommended that you use "Opt-In a Member".
Opt-In a Member
Send an optin email to the specified email address, creating the member if not already existing and adding it to the specified list of groups.
Find Member
Find an existing member.
