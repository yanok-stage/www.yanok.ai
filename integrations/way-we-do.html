Way We Do Integrations

Way We Do is an Active Standard Operating Procedures software, enabling your team to follow repeatable process within their daily workflow.

Triggers

Checklist Step Completed
Triggers when a checklist step is completed.
New Comment
Triggers when a new comment is added to a checklist.
Checklist Started
Triggers when a checklist instance is started.
Checklist Finished
Triggers when a checklist is finished.
Procedure Accepted
Triggers when a procedure has been accepted.
Supervisor SignOff Invite
Triggers when a supervisor is invited for signoff.

Actions

Add a Checklist Comment
Adds a new comment to an Activated Checklist Instance step.
Complete a Checklist Step
Mark a specific step in a checklist instance as complete.
Start Checklist Instance
Starts a checklist instance for a user.
Add a New User
Creates and invites a new user.
Find a Checklist by Title
Search for a checklist by title.

Find a Checklist Step by Title
Search for a checklist step by title.
Find a Checklist Step by ID
Search for a checklist step by ID.
Find a Procedure by Title
Search for a standard procedure by title.
Find a Procedure or Checklist by ID
Search for a procedure or checklist by ID.
Find a User
Search for a user by email or name.
