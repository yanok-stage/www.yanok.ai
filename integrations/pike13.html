Pike13 Integrations

The mobile-friendly Pike13 platform allows class-based businesses like strength and conditioning gyms, yoga studios, and pilates studios to manage everything from clients and schedules to billing and payroll.

Triggers

Event Occurrence Upcoming
Triggers when an Event Occurrence is about to be notified, 24 hours by default.
Invoice Created
Triggers when a new invoice is added to your business.
Invoice Updated
Triggers when an invoice is updated.
Person Created
Triggers when a new person is added to your business.
Person Updated
Triggers when a person is updated in your business.
Transaction Created
Triggers when a new transaction is made.
Transaction Updated
Triggers when a transaction is voided or settled.

Actions

Post Roster Note
Add a new note to an event occurrence.
Create Person
Add a new Person to a Pike13 Business.
Update Person
Update an existing Person in a Pike13 business.
Find Invoice
Find an existing Invoice by ID, Created Since, or Updated Since fields. Use of at least one search field is required.
Find Person
Find an existing person by their Name, Email, or Barcode. Use of at least one search field is required.
Person Search or Create
Search for a person. If the person does not exist, create the person.
