LeadDelta Integrations

LeadDelta is a productivity app that helps you to organize your LinkedIn connections better and save time.

Triggers

Applied Tag
Triggers when a tag is applied to a connection.
New Connection
Triggers when a new connection is added.
New Disconnect
Triggers when a connection is disconnected.
New Follow
Triggers when a connection is followed.
New Message
Triggers when a new message is sent to a connection.
New Note
Triggers when a new note is added to a connection.
New Profile Visit
Triggers when a new profile visit is done on LinkedIn via LeadDelta.
New Unfollow
Triggers when a connection is unfollowed.

Actions

Add Note
Adds a note to a connection.
Apply Tag
Applies a tag to a connection
Create Tag
Creates a new tag
