Ignition Integrations

Ignition makes it simple to engage clients, get paid and run your business on autopilot. From impressive online proposals to automated engagement letters and payments, we free you up to focus on what matters most.

Triggers

Proposal Accepted
Triggers when a proposal is accepted.
Proposal Archived
Triggers when a proposal is archived. Does not work with Classic proposals.
Proposal Completed
Triggers when a proposal is completed. Does not work with Classic proposals.
Proposal Marked as Lost
Triggers when a proposal is marked as lost. Does not work with Classic proposals.
Proposal Moved to Awaiting Acceptance
Triggers when a draft proposal is moved to awaiting acceptance. Does not work with Classic proposals.
Proposal Revoked
Triggers when a proposal is revoked. Does not work with Classic proposals.
Proposal Saved as Draft
Triggers when a proposal is moved to draft state. Does not work with Classic proposals.
Classic Proposal State Changed
Triggers when a proposal's state changes. Only works with Classic proposals
Service Accepted by Client
Triggers for each service in an accepted proposal. As an example, if a client accepts a proposal containing three services, this will trigger three times - once for each of the services.

Actions

Create Client
Creates a new client.
