Steam Integrations

Steam is a digital distribution platform developed by Valve Corporation for purchasing and playing video games.

Triggers

New Owned Game
Triggers when a new game is added to your Steam Library.
New Played Game
Triggers when a game you own is played for the first time.
Time Played Updated
Triggers when the time spent playing a game is updated. This happens whenever the game is exited or every 30 minutes, whichever comes first.
