Inmobalia CRM Integrations

Inmobalia CRM is the online real estate software and secure property MLS network servicing the most recognized real estate agencies on the Costa del Sol, probably one of the most competitive real estate markets in Spain and the whole Europe

Triggers

New Activity
Triggers when a new Activity is created.
New Contact
Triggers when a new Contact is created.
New Property
Triggers when a new Property is created. Use Property Update trigger later to get more details if needed.
New WebLead
Triggers when a new WebLead is registered.
Activity Update
Triggers when an Activity is updated or created.
Contact Update
Triggers when a Contact is updated or created.
Property Update
Triggers when a Property is updated or created.
New User
Triggers when a new User is created.

Actions

Create Activity
Creates a new Activity.
Create Contact
Creates a new Contact.
Update Contact
Updates an existing Contact identified by ID. Use Find Contact by Email action to know the ID of a contact by email address.
Create WebLead
Creates a new WebLead.
Find and Get Activity
Find an Activity by ID and get its details. Use it in conjunction with New Activity trigger.
Find and Get Contact by ID
Find a Contact by ID and get its details. Use it in conjunction with New Contact trigger.
Find and Get Property by ID
Find a Property by ID and get its details. Use it in conjunction with New Property or Property Update triggers.
Find and Get Property by Reference
Find a Property by Reference and get its details.
Find and Get Contact by Email
Find a list of Contacts by email and get its details.
Find and Get User
Find an User by username and get its details.
Find or Create Contact
Find a contact by email address and get its details. If it doesn't exist it creates a new Contact with the data provided.
