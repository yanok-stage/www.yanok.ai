Linkseller Integrations

Linkseller helps teams manage leads and accelerate closure simply and intelligently. A Linkseller é um CRM focado em simplicidade e versatilidade para se ajustar a sua empresa. Ajudamos equipes de vendas a gerenciar leads e acelerar o fechamento dos negócios. E acompanhar a produtividade da sua equipe.

Triggers

Person Created (Pessoa Adicionada)
Triggers when a new Person is created (Executado quando uma Pessoa é adicionada).
My Tasks Created (Minhas Tarefas Adicionadas)
Triggers when a new Task is created to yourself (Executado quando uma Tarefa é adicionada para você).
Opportunities by Stage (Oportunidades Por Etapa)
Triggers when a Opportunity is set on a Stage (Executa quando uma Oportunidade entrar na Etapa escolhida).
Opportunities Created (Oportunidades Criadas)
Triggers when a Opportunity is created (Executado quando uma Oportunidade for criada.
Opportunities Lost (Oportunidades Perdidas)
Triggers when a Opportunity is set as lost (Executa quando uma Oportunidade entrar na Etapa perdida).
Opportunities Won (Oportunidades Ganhas)
Triggers when a Opportunity is set as won (Executa quando uma Oportunidade entrar na Etapa ganha).
Web Leads Created (Web Leads Criadas)
Triggers when a Web Lead is created (Executado quando uma Web Lead for criada).

Actions

Create Task Organization (Adicionar Tarefa - Organização)
Create a Task for a Organization (Adiciona Tarefa para uma Organização)
Create Task Person (Adicionar Tarefa - Pessoa)
Create a Task for a Person (Adiciona uma Tarefa para uma Pessoa)
Create Web Lead (Adicionar Web Lead)
Create a new Web lead for a Campaign (Adiciona uma Web lead para uma Campanha)
