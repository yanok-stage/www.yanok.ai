PassKit Event Tickets Integrations

PassKit Event Tickets lets you manage your event productions and passes for Apple Wallet and Google Pay

Triggers

New Ticket Issued
Triggers when a new ticket is issued for a Production.

Actions

Issue Ticket by Date
Issue a new ticket for the provided date.
Issue Ticket by Event Id
Issue a new ticket for an exiting event.
Redeem Ticket
Redeems a ticket by either internal PassKit ID or Ticket Number.
