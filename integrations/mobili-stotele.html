Mobili Stotelė Integrations

Mobili stotelė is a mobile virtual PBX service for Tele 2 mobile subscribers.

Triggers

Call Completed
Triggers when new call has been completed in SmartMEX platform.
Call Connected
Triggers when new call has been answered in SmartMEX platform.
Call Recorded
Triggers when call recording gets created in SmartMEX platform.
Call Started
Triggers when new call has started in SmartMEX platform.
Get Call History
Triggers when list of calls from call history is requested.
Get Call Records
Triggers when list of call records is requested.
Get Contacts
Triggers when list of all employees is requested.
Get Groups
Triggers when list of employee groups is requested.
Get Lost Calls
Triggers when list of lost calls is requested.
Get Voicemails
Triggers when list of voicemails is requested.
Get Voicemail Boxes
Triggers when list of created voicemail boxes is requested.
LostCaller Added
Triggers when call does not get answered and gets registered as lost in SmartMEX platform.
Lost Caller Removed
Triggers when call gets removed from the lost calls list in SmartMEX platform.
LostCaller Updated
Triggers when call does an existing lost call number gets updated in SmartMEX platform.
Voicemail Created
Triggers when new voicemail has been left in SmartMEX platform.
Voicemail Deleted
Triggers when voicemail has been deleted in SmartMEX platform.
