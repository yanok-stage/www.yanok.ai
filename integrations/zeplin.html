Zeplin Integrations

Zeplin is an organized workspace to publish designs where the entire team can collaborate to ship beautiful products together.

Triggers

New Note or Reply
Triggers when a new note or reply is created.
New or Updated Project Resource
Triggers when a resource such as screen and component is created or updated on a project.
New Screen
Triggers when a new screen is created.
New Screen Version
Triggers when a new screen version is created.
New or Updated Styleguide Resource
Triggers when a resource such as component and color is created or updated on a styleguide.
Updated Project
Triggers when a project is updated.
Updated Screen
Triggers when a screen's name, description or tags field is updated.

Actions

Create or Reply to a Note
Creates a new note or a reply to an existing note.
Update Project
Updates a project's name, description, or status.
Update Screen
Updates a screen's description or tags.
Find Project
Finds a project by name.
Find Screen
Finds a screen by name.
Find Project Status
Finds a project status by name.
